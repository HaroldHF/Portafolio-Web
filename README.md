## 🎯 Perfil Objetivo

El perfil profesional investigado es **DevOps Engineer** con enfoque en infraestructura cloud, automatización y sistemas escalables.

Este rol combina desarrollo de software, administración de sistemas y automatización de infraestructura para permitir despliegues rápidos, seguros y confiables.
